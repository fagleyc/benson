1. Register Benson's Signal Number

One-time. Use a number you control — Google Voice works (Signal accepts VoIP via voice verification).

  1. Open signalcaptchas.org/registration/generate.html in a new tab.
  2. Solve the captcha. When the page asks "Open Signal?", copy the link instead — right-click the "Open Signal" button → Copy Link Address. Paste the whole signalcaptcha://… URL below.
  3. Enter your Google Voice number in E.164 format (e.g. +13035551234).
  4. Click Register. Within ~10s, Signal will voice-call your GV number with a 6-digit code. Answer in the GV app or voice.google.com.
  5. Type the code below and click Verify.
2. Enter Verification Code

On success, add to /etc/benson/env: 

SIGNAL_BENSON_NUMBER=+13035551234

…then sudo systemctl restart benson.

Allowlist

Who can talk to Benson via Signal. Edit /etc/benson/env and restart benson: 

SIGNAL_ALLOWED_NUMBERS=+15551112222,+15553334444
SIGNAL_SPEAKER_MAP=+15551112222:Casey,+15553334444:Lindsey
SIGNAL_ALLOWED_GROUPS=GROUP_BASE64_ID_HERE

All numbers in E.164 (+1…). Group IDs appear in /signal/status after the first message. Anyone not on the allowlist is silently ignored.
